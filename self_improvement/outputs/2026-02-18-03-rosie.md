## What I Built

### `self_improvement/scripts/continuation_check.py`

**Purpose:** At the start of each agent cycle, run this to surface all pending tasks assigned to that agent from TODO.md. Enforces the LOOPS.md anti-pattern "Don't stop work halfway."

**Key features:**

| Feature | Detail |
|---------|--------|
| Agent filtering | `--agent [rosie\|mack\|winnie\|lenny\|all]` |
| Priority grouping | URGENT / HIGH / MEDIUM / LOW / OTHER tiers parsed from headers |
| Output modes | Human-readable (default) + `--json` for machine parsing |
| Urgent-only filter | `--urgent-only` for quick pre-cycle check |
| Exit codes | 0=clean, 1=urgent pending, 2=non-urgent pending |
| CI-ready | Can be used in cron health checks / pre-flight validation |
| Path configurable | `--todo-path` override for testing |

**Architecture:**
1. Parse TODO.md with regex — sections by H2/H3 headers, tasks by `[ ]` pattern
2. Extract agent tag from `**[AgentName]**` markdown
3. Map section headers → priority tiers (URGENT/HIGH/MEDIUM/LOW/OTHER)
4. Skip "Completed", "Discovered", "Blockers" sections (don't count done work as pending)
5. Filter by requested agent, output formatted summary + exit code
