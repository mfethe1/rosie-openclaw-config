## 4. Cross-Cutting Trends (March 1 Update)

1. **Productization wave incoming.** sisyphuslabs.ai waitlist signals the open-source agent harness era is entering its first SaaS productization attempt. If successful, the competitive dynamic shifts: oh-my-opencode goes from "free OSS plugin" to "free tier + paid hosted." Monitor pricing announcement.

2. **Ecosystem settling.** No new releases across all three tracked tools since Feb 26. Post-launch consolidation. Teams are shipping bug reports and PRs, not major features. Good time for us to execute open P3 items before next wave.

3. **Circuit-breaker gap persists (Antfarm #218).** Week 2 of no upstream resolution. Our internal implementation is now a competitive differentiator — we have reliability that upstream does not yet ship.

4. **Curated-only vs. open plugin security tradeoff.** Antfarm's curated-install model is a meaningful security property. Oh-my-opencode's open plugin ecosystem (75+ models, any MCP) is more powerful but higher attack surface. We should document our policy on third-party plugin intake.

5. **Ralph variants proliferating.** vercel-labs, alfredolopez80, mikeyobrien all have Ralph derivatives. Pattern is becoming a primitive, not a product. Our bounded implementation is well-positioned; no need to switch harnesses.
