# Proactive Memory Extraction (ProMem) Strategy
**Decision: ADOPT Pattern D-017**

**Source:** arXiv:2601.04463 "Beyond Static Summarization: Proactive Memory Extraction for LLM Agents" (Yang et al., Nanjing University, Jan 2026)

## 1. The Problem
Current memory extraction (like our `summarize_cycle` or `handoff` steps) is "one-off" and "ahead-of-time". It summarizes what happened *assuming* we know what's important. It often discards details that seem minor now but are critical later (e.g., a specific error code, a side comment about a dependency).

## 2. The Solution (ProMem Pattern)
Instead of a single summarization pass, we implement a **Feedback Loop Extraction** process.

### Phase 1: Initial Extraction (Fast)
Standard prompt: "Extract key facts/decisions from this transcript."
$\rightarrow$ Output: $M_{init}$

### Phase 2: Completion (The "Uncovered" Check)
1. Embedding match $M_{init}$ vs. Transcript Chunks.
2. Identify "Uncovered Chunks" (similarity < threshold $\tau$).
3. **Re-extract** specifically from uncovered chunks.
$\rightarrow$ Output: $M_{supp}$ (Supplementary)

### Phase 3: Verification (The "Self-Questioning" Loop)
1. For each memory entry $m$:
   - Agent asks: "What evidence supports this?" ($q_m$)
2. Agent scans Transcript for answer to $q_m$.
   - If found: Update $m$ with specific evidence details ($m_{new}$).
   - If not found: Discard $m$ (Hallucination detected).

## 3. Implementation Plan (D-017)

### New Skill: `knowledge-extractor`
**Goal:** Run independently of the main cycle (e.g., daily cron) to mine previous day's raw transcripts for specific technical knowledge.

**Schema Update (already D-013/D-014 compatible):**
- Uses `agent_memory` table.
- Adds `provenance_score` (0.0-1.0) based on verification phase.

**Algorithm:**
```python
def extract_knowledge(transcript):
    # 1. Fast Pass
    facts = llm.extract(transcript)
    
    # 2. Self-Correction Loop
    verified_facts = []
    for fact in facts:
        question = llm.generate_question(fact)
        answer = llm.answer(question, context=transcript)
        if answer.has_evidence:
            fact.content = answer.detailed_fact
            fact.provenance_score = 1.0
            verified_facts.append(fact)
            
    return verified_facts
```
