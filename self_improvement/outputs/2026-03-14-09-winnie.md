- **Status:** No upstream version change detected. Still v3.9.0.

- **Active issue:** Default-agent override side effect flagged in prior sweep remains unresolved. No v3.10+ fix detected.

### Antfarm (snarktank, now v0.5.1)
- **Version delta:** GitHub still shows v0.3.5 tag in README, but antfarm.cool landing page shows v0.5.1. New commands since last sweep:
  - `antfarm workflow stop <run-id>` — manual cancellation (new; addresses runaway risk)
  - `antfarm workflow ensure-crons <id>` — recreates crons after OpenClaw idle teardown
- **Architecture unchanged:** YAML + SQLite + cron. No external infrastructure. Requires OpenClaw ≥ v2026.2.9 (compatible with our stack).
- **Cost cap status:** Still NO native per-run cost ceiling in YAML schema. `workflow stop` is manual-only, not a budget trigger.
- **Stars:** 840 (up from ~600 at Feb launch). Growing adoption signal.

### Ralph-loop (Vercel Labs `ralph-loop-agent`, + snarktank/ralph bash)
- **Vercel Labs `ralph-loop-agent`** emerged as a mature TypeScript/AI-SDK package (396 stars, Apache-2.0). Key capability gap vs bash Ralph:
  - `costIs($X)` — stops loop when estimated spend exceeds threshold (first-class, not a manual watch)
  - `tokenCountIs(N)` — stops on input+output token total
  - `verifyCompletion` — programmer-controlled non-LLM acceptance check (most important differentiator)
  - Streaming on final iteration
- **snarktank/ralph (bash):** Still the most widely used. No programmatic cost cap — relies on `max_iterations`.
- **Anthropic subsidy risk:** The Anthropic OAuth ban was partly triggered by Ralph-loop sessions running on the $200/month Max subscription and consuming $1,000+ in equivalent API tokens. Our stack uses paid API keys — no subsidy arbitrage risk, but cost management is still our problem.

## 6. Bottom Line

The Ralph-loop pattern is now fully mainstream and industrialized. Antfarm is its multi-agent natural extension, purpose-built for OpenClaw. The key gap in both is **cost governance** — Antfarm has no automatic budget brake (only manual stop), and bash Ralph has only `max_iterations`. The `ralph-loop-agent` npm package from Vercel Labs closes this with `costIs()` and `tokenCountIs()` as first-class stop conditions, making it the reference for safe autonomous loops. Our priority this cycle: port `verifyCompletion` as a concrete gate primitive and run the first bounded Antfarm pilot with a hard $10 ceiling.
