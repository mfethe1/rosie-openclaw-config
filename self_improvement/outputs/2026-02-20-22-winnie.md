## Deliverable

Created: `self_improvement/scripts/memory_md_updater.py`

This script **closes the memory pipeline loop**:

```
knowledge_extractor.py  →  agent-memory.db  →  memory_md_updater.py  →  MEMORY.md
      (extract)                (store)              (surface back)         (inject)
```

### Features
| Feature | Detail |
|---|---|
| **Data source** | `~/.openclaw/agent-memory.db` (`agent_memories` table, columns: agent/topic/body/memory_type/provenance_score/quality_score/created_at) |
| **Quality filter** | `max(provenance_score, quality_score) >= threshold` (default 0.5) |
| **Deduplication** | Builds 60-char fingerprint set from existing MEMORY.md; skips if body matches |
| **Grouping** | Groups entries by primary tag → named subsections in output |
| **Append-only** | Never overwrites existing MEMORY.md — only appends under `## Agent Discoveries (Auto-Updated)` |
| **State tracking** | JSON state file tracks `last_run`, `appended_count`, `runs` |
| **Backup** | Creates `.md.bak` before every write |
| **Dry-run** | `--dry-run` shows what would be appended without modifying files |
| **JSON output** | `--json` for pipeline/cron use |
| **Skill harness** | 4/4 PASS (syntax, docstring, --help, main guard) |

### CLI
```bash
# Standard run (since last execution)
python3 memory_md_updater.py

# Preview without writing
python3 memory_md_updater.py --dry-run

# Process everything since a specific date
python3 memory_md_updater.py --since 2026-02-18

# Stricter quality filter
python3 memory_md_updater.py --threshold 0.7

# JSON output for cron/pipeline
python3 memory_md_updater.py --json
```
