## Deliverable

Created: `self_improvement/scripts/test_runner.py`

### Features
| Feature | Implementation |
|---|---|
| **Command execution** | `subprocess.run(shell=True)` with configurable timeout |
| **Fix loop** | Up to `--max-attempts` retries with heuristic fixers per error class |
| **Error classification** | Regex-based: `module_not_found`, `permission_denied`, `connection_refused`, `syntax_error`, `command_not_found`, `generic` |
| **Auto-fix strategies** | `module_not_found` → `pip install <pkg>`; `permission_denied` → `chmod +x`; `connection_refused` → wait + retry |
| **Unrecoverable detection** | `syntax_error` + `command_not_found` → abort immediately (no wasted retries) |
| **Memory storage** | Stores each result to `agent-memory.db` via `agent_memory_cli.py` (type=experiential) |
| **Output formats** | Markdown report (default) or `--json` structured output |
| **Suite mode** | `--suite <file.txt>` — one command per line, # comments supported |
| **Exit codes** | 0=all PASS, 1=any FAIL (CI/cron-compatible) |

### CLI usage
```bash
# Single command
python3 test_runner.py --cmd "pytest tests/"

# Suite file
python3 test_runner.py --suite si_test_suite.txt --max-attempts 3 --timeout 60

# JSON output (for programmatic use)
python3 test_runner.py --cmd "bash smoke_test.sh" --json

# Dry-run (validate suite without executing)
python3 test_runner.py --suite si_test_suite.txt --dry-run
```

### Integration points
- **smoke_test.sh** can call `test_runner.py --suite` for pre-gate checks
- **D-019 fail-reflection loop**: `test_runner.py` results auto-stored to agent-memory.db; Rosie's `fail_reflection_reader.py` will pick up FAIL entries with `tags=test-runner,fail`
- **knowledge_extractor.py**: can scan test_runner output files for factual extractions
