```sql
CREATE TABLE IF NOT EXISTS skills (
  name TEXT PRIMARY KEY,
  trigger TEXT,
  inputs TEXT,
  steps TEXT,
  outputs TEXT,
  agent_owner TEXT,
  use_count INTEGER DEFAULT 0,
  last_used TEXT,
  created_at TEXT DEFAULT (datetime('now'))
);
```

### New Commands
- `python3 agent_memory_cli.py skill search <query>`
- `python3 agent_memory_cli.py skill add <name> --trigger ... --steps ...`
