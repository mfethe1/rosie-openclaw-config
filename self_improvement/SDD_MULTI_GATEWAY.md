# Systems Design Document — Multi-Gateway Architecture
Date: 2026-03-01 | Source: Principal Architect (Michael)

## EPIC 3: Rosie's Domain (AppSec, Consensus & Alignment)

### Zero Trust Thesis
Code generated by LLMs is inherently malicious or structurally flawed until mathematically proven otherwise.

### F5 & UB3: Verification Gate & BCCS
- Actor-Critic with 3 Critics at varying temperatures
- Weighted mean: μ = Σ(c_i × w_i) / Σ(w_i) where w_i = historical reliability
- μ ≥ 0.82 → auto-commit to Postgres
- 0.5 ≤ μ < 0.82 → debate (inject lowest Critic rationale back to Actor)
- μ < 0.5 → prune branch
- NATS topic: unverified_actions

### UB9: Sandbox + AST Analyzer
- Docker: USER nonroot, --cap-drop=ALL, --security-opt=no-new-privileges
- AST analyzer: flag os.system, subprocess, eval, exec, __import__
- Scan ClawHub skills before install

### UB7: SOP Encoding
- Convert Markdown SOPs → DSPy-style XML state machines
- Target: 40-60% token reduction
- Standardize rule enforcement
