## 2026-02-27 10:00 EST — Memory/Comms system research findings

### State-of-the-art systems evaluated
1. **Agno/AgentOS**: memory primitives (short-term context + long-term user learning), cross-session state management, enterprise runtime. Most applicable pattern: session-consistent memory + long-running state management.
2. **Zep/Graphiti**: temporal knowledge graph for agent memory. Nodes have temporal validity. Outperforms RAG by 18.5% on accuracy. Key feature: time-aware entity resolution + edge invalidation (supersedes pattern we want).
3. **A-Mem (NeurIPS 2025)**: Zettelkasten-based dynamic memory structuring. Atomic notes + flexible link graph + agent-driven reorganization. Best pattern for memU evolution.
4. **MemRL (Jan 2026)**: decouples stable reasoning (frozen LLM) from plastic episodic memory. Runtime RL updates memory without re-training model. Most novel — enables self-improving memory at runtime.
5. **Distributed shared memory + periodic sync**: industry consensus pattern for multi-gateway consistency. Local state for speed, sync to shared store periodically.

### Priority adoption path for our stack
- P0: Temporal knowledge graph edges in memU (time-aware supersedes/invalidation) — inspired by Zep/Graphiti
- P0: Cross-gateway shared memory access with local cache + periodic sync
- P1: A-Mem Zettelkasten-style atomic memory notes with link graph in memU
- P1: MemRL-style episodic memory separate from reasoning (decouple what agent knows from how it reasons)
- P2: Agno-style session memory primitives for cross-agent session continuity
